🦁 AnimalWorld

AnimalWorld is a web application that allows users to explore different categories of animals such as Mammals, Birds, Fish, Reptiles, Amphibians, and Invertebrates. Users can browse animal information, view detailed descriptions, upload images, add new species, edit existing records, and delete animal entries.

✨ Features
* 🔐 User Authentication (Login & Sign Up)
* 🐾 Browse Animals by Category
* 📖 View Detailed Animal Information
* ➕ Add New Animal Species
* ✏️ Edit Animal Details
* 🗑️ Delete Animal Records
* 🖼️ Upload Animal Images
* 💾 MySQL Database Integration
* 📱 Clean and User-Friendly Interface

🛠️ Technologies Used
* Node.js
* Express.js
* EJS
* MySQL
* HTML5
* CSS3
* JavaScript
* Multer
* UUID
* Method Override

🚀 Installation
Clone the repository : git clone https://github.com/prachi-raut-04/AnimalWorld.git

Install dependencies : npm install

Start the server : node index.js or nodemon index.js

Open your browser : http://localhost:8080
